## 题目

Given a non-negative integer `c`, decide whether there're two integers `a` and
`b` such that `a2 + b2 = c`.

**Example 1:**

> Input: c = 5
>
> Output: true
>
> Explanation: 1 _ 1 + 2 _ 2 = 5

**Example 2:**

> Input: c = 3
>
> Output: false

**Constraints:**

- `0 <= c <= 2^31 - 1`
