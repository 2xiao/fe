## 题目

Given the root of a binary tree and two integers `p` and `q`, return
_the**distance** between the nodes of value _`p` _and value_`q` _in the tree_.

The **distance** between two nodes is the number of edges on the path from one
to the other.



**Example 1:**

![](https://fastly.jsdelivr.net/gh/doocs/leetcode@main/solution/1700-1799/1740.Find%20Distance%20in%20a%20Binary%20Tree/images/binarytree.png)

> Input: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 0
> 
> Output: 3
> 
> Explanation: There are 3 edges between 5 and 0: 5-3-1-0.

**Example 2:**

![](https://fastly.jsdelivr.net/gh/doocs/leetcode@main/solution/1700-1799/1740.Find%20Distance%20in%20a%20Binary%20Tree/images/binarytree.png)

> Input: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 7
> 
> Output: 2
> 
> Explanation: There are 2 edges between 5 and 7: 5-2-7.

**Example 3:**

![](https://fastly.jsdelivr.net/gh/doocs/leetcode@main/solution/1700-1799/1740.Find%20Distance%20in%20a%20Binary%20Tree/images/binarytree.png)

> Input: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 5
> 
> Output: 0
> 
> Explanation: The distance between a node and itself is 0.



**Constraints:**

  * The number of nodes in the tree is in the range `[1, 104]`.
  * `0 <= Node.val <= 10^9`
  * All `Node.val` are **unique**.
  * `p` and `q` are values in the tree.
