# 2928. 给小朋友们分糖果 I

🟢 <font color=#15bd66>Easy</font>&emsp; 🔖&ensp; [`数学`](/tag/math.md) [`组合数学`](/tag/combinatorics.md) [`枚举`](/tag/enumeration.md)&emsp; 🔗&ensp;[`力扣`](https://leetcode.cn/problems/distribute-candies-among-children-i) [`LeetCode`](https://leetcode.com/problems/distribute-candies-among-children-i)

## 题目

You are given two positive integers `n` and `limit`.

Return _the**total number** of ways to distribute _`n` _candies among_`3`
_children such that no child gets more than_`limit` _candies._



**Example 1:**

> Input: n = 5, limit = 2
> 
> Output: 3
> 
> Explanation: There are 3 ways to distribute 5 candies such that no child gets more than 2 candies: (1, 2, 2), (2, 1, 2) and (2, 2, 1).

**Example 2:**

> Input: n = 3, limit = 3
> 
> Output: 10
> 
> Explanation: There are 10 ways to distribute 3 candies such that no child gets more than 3 candies: (0, 0, 3), (0, 1, 2), (0, 2, 1), (0, 3, 0), (1, 0, 2), (1, 1, 1), (1, 2, 0), (2, 0, 1), (2, 1, 0) and (3, 0, 0).

**Constraints:**

  * `1 <= n <= 50`
  * `1 <= limit <= 50`
